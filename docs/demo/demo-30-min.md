# Lumio 30-Minute Demo

Date: 2026-03-11
Audience: Buyer committee, admin stakeholders, solution review group

## Demo objective

Show both the governed workflow chain and the enterprise control plane with enough depth for admins, governance owners, and audit stakeholders.

## Opening framing

- B2B enterprise workspace platform
- Product name: `Lumio`
- OA v1 nine-role foundation
- Okta OIDC-only identity target
- `local_lab` sandbox / preview workspace
- current package is credible as a buyer-facing preview, not a claim of full real-pilot or full production closure

## Suggested agenda

### 1. Workspace shell and role model

- Show role switching
- Show section navigation
- Explain the workspace-first shell

### 2. Governed execution flow

Walk one task through:

1. Request Center
2. Approval Center
3. Operations Console
4. Review Center
5. Audit & Reporting Center

Call out:

- current stage
- next action
- blocker
- waiting-on relationship
- shared evidence continuity

### 3. Members & Access

Show:

- seat scope
- invite lifecycle
- join trace
- role/access boundary changes

### 4. Integration & Readiness

Show:

- Okta OIDC-only readiness checklist
- why-ready / why-not-ready
- environment, connector, and credential posture

### 5. Policy & Governance

Show:

- policy basis
- exception / waiver posture
- decision rationale
- blocker linkage back to policy basis

### 6. Audit & Reporting

Show:

- receipt timeline
- evidence bundle
- export bundle
- role/time filtering

### 7. Organization & Workspace

Show:

- ownership
- admin boundaries
- escalation path
- workspace health

## Recommended close

End with:

- what is already coherent now
- what is intentionally sandbox / preview
- what is deferred without overclaiming
