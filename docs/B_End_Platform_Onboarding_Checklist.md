# Lumio Onboarding Checklist

Date: 2026-03-10
Owner: Codex
Scope: Lumio enterprise workspace platform preview

Product name: `Lumio`

## Before walkthrough
- Confirm canonical Lumio preview URL is used.
- Confirm the main demo path uses only the Lumio enterprise workspace preview route.
- Confirm local_lab or current workspace context is intentional.

## Admin onboarding
- Explain Organization & Workspace.
- Explain Members & Access.
- Explain Integration & Readiness Center.
- Demonstrate invite / assign / remove / suspend / reactivate.

## Reviewer / approver onboarding
- Explain Approval Center vs Review Center.
- Demonstrate grouped items, search, filters, and detail expansion.
- Demonstrate one review action and one approval action.

## Auditor onboarding
- Explain Audit & Reporting Center.
- Show receipt / timeline / membership trail grouping.
- Show where audit export records appear.

## Success criteria
- New stakeholder can identify the purpose of each page.
- New stakeholder can follow the member lifecycle.
- New stakeholder can explain the difference between approval, review, and audit.
