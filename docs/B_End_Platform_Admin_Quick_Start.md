# B-End Platform Admin Quick Start

Date: 2026-03-10
Owner: Codex
Scope: B-end platform only

Product name: `Lumio`

## Audience
- Tenant Admin
- Workspace Admin
- Integration Admin

## Primary URL
- Use the B-end platform URL, not the C-end compatibility surfaces.

## First 5 actions
1. Open `Workspace Overview` and verify activation / blocker posture.
2. Open `Organization & Workspace` to confirm environment and workspace facts.
3. Open `Members & Access` to inspect seats, members, and role assignments.
4. Use `Enterprise membership admin` to invite or manage one member.
5. Open `Integration & Readiness Center` to review identity, connector, vault, and activation posture.

## What each admin role owns
- `TENANT_ADMIN`: tenant-level membership, activation confirmation, promote-critical actions.
- `WORKSPACE_ADMIN`: workspace-scoped membership and seat operations.
- `INTEGRATION_ADMIN`: environment / connector readiness and related promotion scope.

## Success state
- Workspace and environment facts are visible.
- Members and role assignments are visible.
- Admin actions are available only when the session is authorized.
