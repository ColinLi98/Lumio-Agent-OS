## 一、执行摘要

在巨头阴影下，以巧劲撬动未来。

**核心矛盾**：用户在数字世界产生的数据越来越多，但这些数据散落在各个平台，既无法掌控，也难以利用。

**Lumi 定位**：从"输入"这一最高频、最基础的场景切入，成为用户在数字世界中处理信息、表达自我的"第二大脑"。

**产品形态**：**智能输入面板** — 不取代现有输入法，而是在用户需要时提供增强能力。

**核心差异化**：
- 🔐 **本地优先**：拒绝云端数据收集，用户数据加密存储在个人设备
- 🔗 **跨应用工作**：在任意 App 的输入场景都能使用
- 🧠 **渐进式身份构建**：使用过程中自然积累个人知识图谱

### 2.2 核心架构

```
用户现有输入法（搜狗、百度等）
    ↓

    ↓
用户本地加密数据库（个人知识图谱）
```

## 三、三大核心功能 (MVP)

### 3.1 帮我写（解决表达焦虑）

**场景**：回复工作消息、写朋友圈文案、邮件起草等

**用户故事**：
> 作为一个职场人，我想要快速生成得体的回复，这样我就不用花时间纠结措辞了。

**工作流程**：
1. 用户在任意 App 的输入框**长按空格**，唤出 Lumi 面板
2. 选择"帮我写"功能
3. Lumi 分析当前聊天上下文（仅限输入框可见内容）
4. 提供 **3-5 种不同风格**的回复建议（专业/幽默/简洁等）
5. 用户选择后**一键填入**

**输出格式**：
```typescript
interface WriteResult {
  drafts: Array<{
    id: string;
    text: string;
    style: 'professional' | 'casual' | 'humorous' | 'concise' | 'warm';
    confidence: number;
  }>;
}
```

**隐私设计**：上下文分析完全在**设备端**进行，不上传任何对话历史。

---

### 3.2 帮我找（解决信息割裂）

**场景**：在小红书看到好物想比价、在公众号看到书想购买、在聊天中收到地址想导航

**用户故事**：
> 作为一个消费者，我看到某个商品后想快速比价，这样我就能做出更明智的购买决定。

**工作流程**：
1. 用户在当前屏幕唤出 Lumi 面板
2. 选择"帮我找"功能
3. Lumi 通过 **OCR（设备端）** 提取屏幕关键信息
4. 提供智能操作：
   - **比价** → 跳转电商搜索
   - **收藏** → 保存到 Lumi 本地库
   - **搜索** → 用默认搜索引擎
   - **导航** → 打开地图应用

**输出格式**：
```typescript
interface FindResult {
  extractedInfo: {
    type: 'product' | 'book' | 'location' | 'link' | 'text';
    content: string;
    confidence: number;
  };
  actions: Array<{
    type: 'compare_price' | 'save' | 'search' | 'navigate' | 'open_link';
    label: string;
    actionUri: string;
  }>;
}
```

**隐私设计**：OCR 完全在**设备端**运行，提取的信息仅用于本次操作，**不存储**。

---

### 3.3 帮我记（解决信息过载）

**场景**：聊天中约定的会议时间、文章中的关键观点、商品链接稍后查看

**用户故事**：
> 作为一个忙碌的人，我想要快速保存重要信息到合适的地方，这样我就不会遗忘任何事项。

**工作流程**：
1. 用户选中文本或唤出 Lumi 面板
2. 选择"帮我记"功能
3. Lumi 解析信息类型（时间/地点/任务/链接等）
4. 提供一键保存：
   - **添加到日历**
   - **创建待办**
   - **保存到笔记 App**（需用户授权连接）
   - **稍后阅读**

**输出格式**：
```typescript
interface RememberResult {
  parsedInfo: {
    type: 'event' | 'task' | 'note' | 'link' | 'contact';
    title: string;
    datetime?: string;
    location?: string;
    url?: string;
    tags: string[];
  };
  suggestedActions: Array<{
    type: 'calendar' | 'todo' | 'note' | 'bookmark';
    label: string;
    targetApp?: string;
  }>;
}
```

**隐私设计**：所有解析在**本地完成**，外部 App 连接通过**系统标准 API**，Lumi 不中转数据。

## 四、硬性约束 (Non-negotiable)

### R1. 绝对禁止误发 (No Mis-send)
- 在 **Agent Mode** 下，用户输入的命令**不得自动提交**到宿主 App 输入框
- 只有用户**明确动作**才允许提交：
  - 点击某条 Draft
  - 点击"确认发送"按钮

### R2. 本地优先与隐私 (Local-first Privacy)
- 默认**不上传**原始聊天内容/输入内容
- 仅在以下条件**同时满足**时允许联网：
  - 处于 Agent Mode
  - 意图需要在线服务
  - Policy 允许
- 联网仅发送：**意图 + 约束**，不发送聊天历史/明文上下文

### R3. Agent Core 必须独立
- Agent Core 必须是独立模块，可在不依赖 UI 的情况下运行与测试
- UI 与 Agent Core 的唯一耦合点为稳定接口：
  - `LumiAgent.handle(input) -> output`

### R4. 性能
- 普通打字路径**不得调用**网络/大模型/重计算
- Agent 处理必须**异步**，不阻塞 UI 线程

### R5. V1.0 禁止自动化操作
- 第一版不实现自动点击/自动填表（默认 OFF、后续版本再评估）

## 五、数字身份的渐进式构建

随着用户使用，Lumi 会在本地构建一个加密的**个人知识图谱**：

| 维度 | 数据来源 | 用途 |
|------|---------|------|
| **兴趣图谱** | 通过"帮我找"记录的兴趣商品、文章主题 | 个性化推荐 |
| **社交风格** | 通过"帮我写"学习的表达偏好 | 文风匹配 |
| **时间模式** | 通过"帮我记"积累的日程规律 | 智能提醒 |

**用户控制权**：
- ✅ 可随时**查看**所有数据
- ✅ 可随时**编辑**或**删除**
- ✅ 可随时**导出**完整数据
- ✅ 可一键**清空**所有数据

// Agent 输入
interface AgentInput {
  rawText: string;
  mode: 'TYPING' | 'AGENT';
  feature: 'WRITE' | 'FIND' | 'REMEMBER';
  appContext?: {
    appName: string;
    fieldType?: string;

  | { type: 'DRAFTS'; drafts: TextDraft[] }

// Soul Matrix - 个性化配置
interface SoulMatrix {
  communicationStyle: 'professional' | 'casual' | 'warm' | 'humorous';
  riskTolerance: 'low' | 'medium' | 'high';
  privacyLevel: 'strict' | 'balanced' | 'relaxed';
}

// Policy 配置
interface PolicyConfig {
  allowNetworkInAgentMode: boolean;
  requireConfirmBeforeSend: boolean;
  allowedServices: string[];
}

## 八、验收标准 (MVP)

### AC1. No Mis-send
在任意 App 输入框（微信/浏览器/备忘录）：
- ✅ 进入 Agent Mode
- ✅ 输入命令并按回车
- ✅ 宿主输入框**不得出现**该命令
- ✅ 仅点击 Draft 或确认发送后才会提交

### AC2. 帮我写 可用
- ✅ "帮我委婉拒绝加班" => 输出 3 条不同风格的草稿
- ✅ "写一条生日祝福" => 输出 3 条草稿
- ✅ 草稿风格符合 SoulMatrix 设置

### AC3. 帮我找 可用
- ✅ 屏幕上有商品名 => 识别并提供比价链接
- ✅ 屏幕上有地址 => 识别并提供导航选项
- ✅ OCR 在设备端运行，不上传图片

### AC4. 帮我记 可用
- ✅ 选中"明天下午3点开会" => 识别为日程，提供添加到日历
- ✅ 选中链接 => 提供"稍后阅读"保存选项
- ✅ 保存的内容可在本地查看和管理

### AC5. 离线可用
开启飞行模式：
- ✅ "帮我写"仍可输出 Drafts（本地模板）
- ✅ "帮我找"返回离线提示，不崩溃
- ✅ "帮我记"正常保存到本地
